# ⚡ SmartGen - Premium QR Code Generator

SmartGen is a fast, free, and highly customizable QR code generator tool. Designed for modern creators, businesses, and developers, it allows you to create professional-grade QR codes for various purposes with real-time previews and high-quality exports.

🔗 **Live Demo:** [qr.genzfrontir.com](https://qr.genzfrontir.com)

---

## 🚀 Key Features

- **Multi-Purpose Generation:** Create QR codes for:
  - 🌐 **URL:** Link to any website or social profile.
  - 📶 **WiFi:** Share your network credentials securely.
  - 💬 **WhatsApp:** Pre-filled messages for easy contact.
  - 📧 **Email:** Subject and body templates.
  - 📱 **SMS:** Instant text messaging.
  - 👤 **vCard:** Digital business cards.
- **Premium Customization:**
  - Change Foreground and Background colors.
  - Choose from different Dot Patterns (Square, Dots, Rounded).
  - Customize Corner Square and Corner Dot styles.
- **Real-Time Preview:** See your changes instantly as you type.
- **High-Quality Downloads:** Export your codes in **PNG** and **SVG** formats.
- **Privacy Focused:** 100% secure, generating codes directly in your browser.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop users.

---

## 🛠️ Built With

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties and smooth UI animations.
- **Vanilla JavaScript:** Fast and lightweight logic.
- **QRCode.js:** Reliable QR generation engine.
- **Google Analytics (GA4):** For user insights and traffic monitoring.

---

## 📖 How to Use

1. Clone or download the repository.
2. Open `index.html` in any web browser.
3. Select your desired QR type (URL, WiFi, etc.).
4. Enter your data and customize the colors/styles.
5. Click **Download PNG** or **Download SVG** to save your QR code.
